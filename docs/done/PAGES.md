# 페이지 목록

사이드바 네비게이션 순서(영업 / 정산 / 관리)를 기준으로 정리.
파일 경로는 `src/pages/` 하위 flat 구조, 네이밍은 `XxxPage.vue`.

## 영업

| 파일                   | 라우트            | 설명                                                                                                                 |
| ---------------------- | ----------------- | -------------------------------------------------------------------------------------------------------------------- |
| `OrderPage.vue`        | `/orders`         | 매장 선택 → 메뉴 담기 → 주문 생성. POS 메인 화면.                                                                    |
| `OrderMonitorPage.vue` | `/order-monitors` | 진행 중 주문/완료 주문 실시간 현황. 경과시간별 색상 변화 (15/25/35분). 완료 처리 · 완료 취소 지원. 주방용 모드 토글. |

## 정산

| 파일                  | 라우트         | 설명                                                                               |
| --------------------- | -------------- | ---------------------------------------------------------------------------------- |
| `SettlementsPage.vue` | `/settlements` | 일/주/월 단위 매출 집계. 현금/카드/미수 KPI + 주문 건별 테이블. 미수 건 결제 처리. |

## 관리

| 파일                  | 라우트         | 설명                                                              |
| --------------------- | -------------- | ----------------------------------------------------------------- |
| `SalesPage.vue`       | `/sales`       | 기간/매장 필터 기반 과거 주문 내역 조회. 상세 보기 · 수정 · 삭제. |
| `ExpensesPage.vue`    | `/expenses`    | 지출 항목(식자재 구매, 공과금 등) 등록 및 조회.                   |
| `ProductsPage.vue`    | `/products`    | 메뉴에 사용되는 제품 마스터 관리 (단가 · 분류 · 원가).            |
| `IngredientsPage.vue` | `/ingredients` | 원재료 관리. 제품과 연동.                                         |
| `MenusPage.vue`       | `/menus`       | 판매 메뉴 등록/수정. 약어(abv) · 분류 · 가격.                     |
| `StoresPage.vue`      | `/stores`      | 매장 등록/수정. 즐겨찾기 · 분류 태그(중앙/농협/원예 등) · 비고.   |
